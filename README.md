# Resonance
A Python package library for creating AI entities that develop identity over time through memory, context, and adaptive personality modeling.

## Features


### Identity Core
The Identity Core defines the fundamental nature of the character.



## THIS PYHON PACKAGE IS STILL IN DEVELOPMENT AND NOT MEANT TO BE USED YET

## Features still worked on

### Memory

The memories and contexts are stored in an SQL database.\
Working on different memory aspects;\
    
- Episodic -> past conversations
- Semantic -> Facts about the world/user
- Emotional -> How the AI feels about things
- Relationship -> Evolving Bond with Users


### Perception Layer
Interpretation of the user's input (emotional state of user, what they mean, what is their intent).

### Internal State
AI's current condition.\
Changes over time\
Influenced by interactions\


### Planning Layer
Plan the best outcome to do in order to better respond to the input.


### Reflection Loop
Analyze AI's output, reflect on it and consider better alternatives for future.
